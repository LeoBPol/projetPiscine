Navigation sur le site

Ce site vous permet de réserver votre créneau de soutenance.
Si vous êtes Administrateur vous avez la possibilité de créer des événements, pour cela rendez-vous dans Créer un événement en cliquant sur le bouton "Créer un événemnet". Vous pourrez à l'issu du formulaire proposer des créneaux pour que les étudiants s'inscrivent.
Si vous êtes étudiant, vous pouvez réserver un créneau sur l'événement qui vous est proposé. Cliquer sur "Reserver votre créneau" et remplissez le formulaire, que l'on puisse connaitre les membres de votre groupe. A l'issue de ce formulaire vous pourrez choisir la date et l'heure de votre soutenance parmis les créneaux proposés par l'administrateur.
